# AI-Powered Student Query Chatbot

## Project Description
This project is an AI-powered chatbot developed using IBM Watson Assistant concepts. The chatbot is designed to answer common student queries related to admissions, examinations, fees, and placements using Natural Language Processing (NLP).

## Technologies Used
- Artificial Intelligence
- Natural Language Processing (NLP)
- Machine Learning
- IBM Watson Assistant
- IBM Watson Studio
- Chatbot Technology

## Features
- Instant responses
- 24/7 availability
- User-friendly interface
- Multiple query handling
- Reduced manual effort
